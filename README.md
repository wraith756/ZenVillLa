## 🚀 About the Project  

Zenvilla is a full-stack real estate platform designed to provide seamless property discovery with fast performance, SEO optimization, and scalable cloud infrastructure.

---

## ✨ Features  

- 🔍 Advanced property search with filters (price, location, type)  
- 🏘️ Detailed property listings with images  
- ⚡ SEO-friendly rendering using Next.js  
- 📱 Fully responsive design  
- ☁️ Cloud deployment on AWS  
- 🗄️ Efficient database handling using PostgreSQL + Prisma  

---
